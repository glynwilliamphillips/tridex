Tridex introduction

desktop/tridex/introduction/introduction.odt

Tridex will be a program to run on the raspberry pi to present 3D crosswords to blind people via an audio microphone and loudspeaker/earphones etc.

The program will not be written in the programming language python. It will be in a combination of HTML and Javascript.

Blind/partially sighted people are the initial target audience as they may be more adept at the 3D visualisation.

2D coordinated are usually expressed as x & y. Extensions to 3D vary in different contexts. For this reason my 3D model is A across, B behind and C Climb. Imagine you are looking at this picture from the left.

We have a starting list of clues and answers. They are from the Cambridge News so serve only as an example for development.

7 letters: The development work is being done assuming a 7 x 7 x 7 cube. This produces 9 2D crosswords clicing in each direction for 3 crosswords.
